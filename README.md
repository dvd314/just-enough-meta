# Just Enough Meta

A small Obsidian plugin for embedding structured YAML metadata blocks directly into notes with customizable rendering

![Demo](assets/demo.gif)

## Why?

LLMs work well with structured data, but raw YAML blocks are often inconvenient to read while writing notes

This plugin allows embedding structured metadata blocks directly into text while rendering them in a more compact and readable way for humans

## Features

- Render structured YAML metadata blocks directly inside notes
- Support for arrays and nested objects
- Human-readable rendering for AI-oriented structured data
- Per-block rendering configuration using `_config`
- Configurable metadata visibility using `include` and `exclude`

## Installation

Extract the archive into:

`.obsidian/plugins/just-enough-meta`

See [examples](docs/examples.md) for more metadata configurations and edge cases.
